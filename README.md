# Psychology Clinic Website

This repository contains an old course project developed for a web design class. The project is a simple PHP-based website for a psychology clinic and was created for educational purposes to practice basic web development concepts.

## Project Overview

The website includes a homepage, article/detail pages, a shared theme structure, and a basic admin panel. It also uses a MySQL database connection and includes an SQL file for the project database.

The main goal of this project was to improve skills in:

- PHP
- HTML and CSS
- MySQL database connection
- Page layout and content organization
- Basic admin panel logic

## Features

- Homepage for the clinic website
- Content/article display page
- Reusable theme/template structure
- Basic styling with CSS
- MySQL database connection
- Admin panel files for managing content
- Included SQL database export

## Project Structure

- `index.php` - main homepage
- `makale.php` - article/detail page
- `Tema.php` - shared theme and layout functions
- `tema1.php` - static page/template version
- `baglanti.php` - database connection
- `css/tema.css` - stylesheet
- `fotolar/` - image assets
- `veritabanı/psikolog_sayfasi.sql` - database export
- `yonetim/` - admin panel files

## Technologies Used

- PHP
- HTML
- CSS
- MySQL

## Notes

This project was prepared as a student assignment, so the code reflects a learning-stage structure and development style. It is being shared on GitHub as an academic and portfolio project.

## Setup

To run this project locally:

1. Place the project folder inside a local PHP server environment such as XAMPP.
2. Import the `veritabanı/psikolog_sayfasi.sql` file into MySQL.
3. Update the database settings in `baglanti.php` if needed.
4. Start Apache and MySQL.
5. Open the project in your browser.
